# 欢迎来到**openJiuwen**社区论坛！

我们很高兴您加入**openJiuwen**社区论坛。openJiuwen社区论坛是由广大的开发者、用户、合作伙伴一起建立的分享、学习平台。在这里，我们可以互相协助解决问题、分享技术经验。论坛参与者出自他们热爱和兴趣到访社区，并为大家提供帮助。对于需要更具体的专业知识或在周末或假期提出的问题，可能需要一些耐心。

## 1. 基本原则

* 我们鼓励互助、互动，但请保持友善交流，不可进行人身攻击，尊重社区的任何一个成员，讨论问题时请尽量做到有理有据，请尽量提供合理的回复来说服别人，切勿令冲突升级。
* 请规范使用用户名、头像、个人资料和签名，不得冒充**openJiuwen**官方人员身份，不得使用容易引起混淆的名称、头像和内容发布信息。
* 应遵守法律法规，避免宗教、政治或其他不恰当的言论，禁止钓鱼、滥发消息、骚扰或造谣等行为。
* 请勿灌水、发广告、刷屏等openJiuwen社区成员都不喜欢的行为。
* 参与社区互动其实也是一个社交活动，需要个人把握社交分寸，遵守基本的社交礼仪，不可接受的参与者行为包括但不限于：
  * 讨论问题上升到人身攻击，挑衅、侮辱或贬低性评论；
  * 公开或私下骚扰；
  * 未经允许发布他人私人信息；
  * 未经允许发布广告或者其它不良信息；
  * 无故刷屏刷帖从而占用公共资源等。
* 如违反以上社区公约，版主或管理员将有权处理：
  * 出现第一次，【警告】并禁言1天；
  * 出现第二次，禁言30天；
  * 出现第三次，停用账号。
  
## 2. 论坛帖子分类

在您学习、测试、调研、开发或者以其他方式使用openJiuwen及相关项目时，如果您遇到问题、有使用经验想要分享，或希望与其他开发者交流，欢迎在以下分类中选择最适合的方式发帖：

 * **各技术板块**：针对具体技术问题，可在对应技术板块发帖，便于精准交流与分享。
 * **问题求助 Help**：如果您在使用过程中遇到问题，可以通过问题求助 Help 进行求助和交流。
 * **经验分享 Tech Blogs**：如果您有使用经验想要分享，可以通过 Tech Blogs 进行分享。

为了让大家技术交流更加高效顺畅，我们建议您发帖时先搜索历史帖子避免重复；分享时提供可复现的代码/实验数据；讨论后若问题解决，请回帖说明或给予肯定，帮助更多人。

期待大家在本技术论坛中收获知识、创造价值！

### 问题求助发帖注意事项

* 在提问前，您可以先搜索官网中的相关教程或论坛中相关的已解决问题和技术干货帖子，参考以上内容进行解决。如果都不能解决您的问题，请在论坛中提问题求助帖子进行解决。
* 问题求助发帖，请注意以下信息。
  * 首先请以具体的报错给帖子起标题，并根据报错信息选择合适的帖子分类，可参考上面论坛帖子分类的介绍选择正确类别。
  * 在帖子正文中请提供相关的环境信息、脚本信息和报错信息，尽可能详细、清晰、正确、精简地描述您的问题。
* 问题闭环，如果您参考解答的回复将问题解决了，那么您需要对问题选择**最佳答案**，至此该问题闭环；如果回复解答不能解决您的问题，请及时补充您问题的最新状态，方便继续分析和定位，直到问题解决并闭环。

### 经验分享发帖注意事项

* 文章内容的要求
  * 内容原创：技术干货都必须是**原创**（原创！原创！）希望大家不要从别的地方搬运和抄袭，如是分享贴，请获得原作者同意后再进行搬运分享，并标注原作者。
  * 内容形式：技术干货的形式可以是多元的，跟人工智能、机器学习、深度学习、openJiuwen等相关，我们支持一切自由形式的原创技术干货！举例但不限于如下三类：
    1. 使用openJiuwen时遇到了xx问题并成功解决该问题，总结问题描述和解决方案；
    2. 参与研究openJiuwen相关项目的技术经验总结；
    3. 某一阶段学习的知识框架体系的总结与实践。
  * 篇幅大小：篇幅大小尽量超过1000字哈，希望各位开发者把内容写的越详细越好！
* 编辑事项
  * 标题确定：标题要清晰明了，涵盖内容的大意，方便其他开发者参考或搜索时能更快找到。
  * 类别选择：分类要根据各自分享的内容进行选择，可参考上面论坛帖子分类的介绍；也可以分享一些前沿的技术，如果没有可选的对应类别，也可以选择‘其他干货’这一项。

## 3. 如何参与贡献openJiuwen论坛

* 回答问题：在论坛中，想要建立自己的“个人品牌”，回复他人问题便是一个建立“好感度”的好方式。当他人发布问题时，你的回复与助力，将给他带来满满的慰藉及好感。帮助他人不止是一种美德，而你也可以在此过程中，提前学习自己未遇到的：bug 处理方式、问题定位、不同场景运维方案等，让自己获得更多技能的提升的可能。
* 撰写技术干货：openJiuwen论坛为广大开发者提供技术文章平台，优秀技术文章将被同步转载至openJiuwen官网技术博客及openJiuwen微信公众号平台进行推广。
